# Lukkey Embedded

This repository is reserved for the embedded firmware source code of the Lukkey hardware wallet.

## Open Source Notice

Lukkey plans to open source the embedded code for the Lukkey hardware wallet on **July 5, 2026**.

The initial release will focus on the firmware components required for transparent review, technical integration, and long-term community collaboration. Until the source code is published, this repository serves as the official public placeholder for the upcoming embedded code release.

To learn more about Lukkey and the Lukkey hardware wallet, visit the official website: https://lukkey.com
